# CSS パターン集

ConiferFruits で使える 5 つのデザインスタイル見本。各ページに同名の .css が紐付いて自動適用される。

## 一覧

- [1. コーポレート](./1-コーポレート.md) — Confluence風、業務文書向け
- [2. ターミナル](./2-ターミナル.md) — ダーク、開発者向け
- [3. エディトリアル](./3-エディトリアル.md) — マガジン、読み物向け
- [4. ポップ](./4-ポップ.md) — カラフル、カジュアル
- [5. ミニマル](./5-ミニマル.md) — モノクロ、タイポ重視

## カスタマイズ

各 .css をコピーして自分のフォルダに置けばそのフォルダ配下に適用される。
編集も可能。
