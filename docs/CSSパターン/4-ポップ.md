## インストール方法

```bash
# 最新版にアップデート
npm install @conifer/ui@latest

# または yarn を使う場合
yarn upgrade @conifer/ui
```

> 🚀 **注目**: v3.0 への移行ガイドは別途公開予定。破壊的変更は最小限です。

新しいコンポーネントの使い方はシンプル。`<Button variant="pop">` 一発でこのスタイルが適用されます。

1. パッケージをアップデート
2. import パスを確認（変更なし）
3. 新しいプロップを好みで追加
4. 楽しむ！
