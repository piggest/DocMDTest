# カレーの作り方

Docusaurus でドキュメントを追加するサンプルページ。

## 材料（4人分）

| 食材 | 分量 |
|------|------|
| 玉ねぎ | 2個 |
| 人参 | 1本 |
| じゃがいも | 2個 |
| 牛肉 | 300g |
| カレールー | 1/2箱 |
| 水 | 800ml |

## 手順

1. 玉ねぎをみじん切り、人参とじゃがいもは一口大に切る
2. 鍋に油をひき、牛肉を炒める
3. 野菜を加えてさらに炒める
4. 水を入れて20分煮込む
5. 火を止めてルーを溶かす
6. 弱火で10分煮込んで完成

:::tip ポイント
ルーを入れた後は沸騰させない。とろみが逃げる。
:::

## コードサンプル

```python
def make_curry(ingredients: list[str]) -> str:
    for item in ingredients:
        print(f"{item}を投入")
    return "完成"
```
